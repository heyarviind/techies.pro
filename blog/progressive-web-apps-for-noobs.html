Progressive web apps are the web apps which look like native apps. PWAs uses latest browserâ€™s APIs to extend their features and provide a native app like experience.

...and what is Service Worker?

Service workers are the worker scripts which works in background for the web app. They are very powerful technology behind PWAs and handle push notifications, cache web app's content, make web app to work offline and lot more

In other words, service workers are just a javascript file that works independent of webapp and in background and it is up to you to write code for notification, content caching, etc

Service workers are available on Android with chrome 50 or higher and not supported by some major browsers but developers are working to bring it on different platforms and browsers.

Appshell

Appshell is not a framework or library, itâ€™s just a design approach that developers can choose to adhere to that is enhanced by the caching abilities of service workers. It is loaded at the beginning of the page load and the content is loaded afterward.

Service worker caches the appshell and makes the web app load quicker after the first load. This improves the overall app experience.

Installability and App Manifest

One of the best features of PWAs is user can install the app just from the browser. Before installed PWA was placed on the home screen but now user can install them just like native apps and will appear in apps drawer too. Like native apps, you can uninstall the PWAs too.
